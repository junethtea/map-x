## 📖 About The Plugin

Multi View (formerly Dual View Pro) is a powerful spatial comparison tool that allows users to compare multiple datasets (e.g., RSRP, SINR, Throughput, and Basemaps) side-by-side without the hassle of toggling layers. Whether you are analyzing Drive Test logs, correlating network quality, or generating multi-parameter reports, this tool dramatically speeds up the optimization workflow.

## ✨ Key Features

* **🎛️ Dynamic Layout Engine:** Seamlessly switch between Dual (2), Triple (3), and Quad (4) map views on-the-fly using draggable splitters for a fully customizable workspace.
* **🔗 Bi-Directional Live Sync:** Panning and zooming in one map instantly synchronizes across all active canvases, ensuring you never lose spatial context during site tracking.
* **📊 Intelligent Statistical Legend:** An auto-generating, compact legend that calculates real-time data distribution (Sample Count & Percentage) based strictly on the active canvas extent and priority layers.
* **🥪 Smart Layering (Z-Order):** Automatically forces critical physical infrastructure data (Sites, Gcells) to the top layer, ensuring they are never obscured by drive test data or coverage polygons.
* **📏 Dynamic Scale Text:** Real-time scale indicator directly under each map canvas, perfect for taking accurate screenshots for reporting.
* **🎯 Zoom to Layer Target:** Instantly center your view on active data layers while smartly ignoring basemaps.
* **🖥️ Ultra-Minimalist UI:** Designed to maximize map visibility with compact navigation tools, 1px splitters, and a quick-maximize toggle for deep-dive analysis.

## 🚀 Installation

1. Download the latest release `.zip` file from this repository.
2. Open QGIS.
3. Go to **Plugins** > **Manage and Install Plugins...**
4. Select **Install from ZIP** on the left panel.
5. Browse to the downloaded `.zip` file and click **Install Plugin**.
6. Ensure the plugin is checked/activated in the **Installed** tab.

## 💡 Usage Guide

1. Click the **Multi View** icon in the QGIS Toolbar to open the command center.
2. Use the **⊞ (Layout)** button on Map 1 to switch between Dual, Triple, or Quad views.
3. Click the **Layer** button on any map to select which data layers to display.
4. Toggle the **S (Sync)** button to lock the map extents together.
5. Toggle the **L (Legend)** button to show or hide the real-time statistical legend.
6. Click the **⌖ (Target)** button to instantly zoom to your selected data layers.
7. Click the **▢ (Maximize)** button to focus on a single map or restore the layout.
